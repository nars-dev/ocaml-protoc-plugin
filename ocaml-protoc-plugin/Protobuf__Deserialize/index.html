Deserialize a reader.

val deserialize : (Base.int * decoder) Base.list -> Protobuf.Reader.t -> Base.unit result

Deserialize takes a list of field sentinals and a reader for the data. Data is decoded into a list of tags, fields and the appropiate function for the tag is called. After all fields have been processed, data can be read from the sentinals, created by the caller
